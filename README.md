## Featured Projects

### Do It Now — Gamified Productivity Tracker
*Kotlin · Jetpack Compose · Firebase · Gemini AI · WorkManager · Room*

A full-featured Android productivity app built around a role-playing game loop. Users set goals (called missions), earn XP on completion, level up a pixel-art character, and maintain daily streaks. The app includes SENSEI — an AI coaching assistant powered by Gemini that reads your live goal state, XP, and level to deliver personalized accountability.

Built with MVVM, offline-first Room DB (7 schema migrations), Firebase Realtime Database for a live global leaderboard, and a 5-worker WorkManager pipeline for time-critical push notifications.

[View Repository](https://github.com/Karaniniiitd/do-it-now-motivation-tracker-app)
