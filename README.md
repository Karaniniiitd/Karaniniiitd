## Featured Projects

### Do It Now — Gamified Productivity Tracker
*Kotlin · Jetpack Compose · Firebase · Gemini AI · WorkManager · Room*

A full-featured Android productivity app built around a role-playing game loop. Users set goals, earn XP on completion, level up a pixel-art character, and maintain daily streaks. Includes SENSEI — an AI coaching assistant powered by Gemini that reads live goal state, XP, and level to deliver personalised accountability. Built with MVVM, offline-first Room DB (7 schema migrations), Firebase Realtime Database for a live global leaderboard, and a 5-worker WorkManager pipeline for time-critical push notifications.

[View Repository](https://github.com/Karaniniiitd/do-it-now-motivation-tracker-app)
